# CRM Plus Store

Estrutura inicial da vitrine pública do ecossistema CRM Plus. O projeto apresenta seis sistemas simples para pequenas empresas, com páginas individuais preparadas para SEO.

## Aplicativos

- **Atlas** — atendimento, agendamento e ordens de serviço para oficinas, sem controle de estoque.
- **Artemis** — cardápio digital, pedidos e caixa para restaurantes.
- **Poseidon** — carteira de clientes, negociações e funil de vendas.
- **Hercules** — checklists, rotinas e auditorias visuais.
- **Pandora** — pesquisas de satisfação e acompanhamento de resultados.
- **Ares** — orçamentos, cotações e pedidos.

## Executar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Escopo desta fase

Esta versão usa somente conteúdo local. APIs, banco, credenciais e autenticação serão definidos em uma fase posterior.
